FarrWebMetaSearch (FARR2 plugin) - 0.9.0 - 29. December 2009

Author: Philipp Tschannen

Description

FarrWebMetaSearch is a generic web retrieval FARR plugin that show results in list mode. It is therefore targeted at web sites delivering list-like results. Results are selected using regular expressions.

Download & Install

Download the FarrWebMetaSearch plugin from here: http://farrplugins.objecttechnology.com/farrwms/FarrWebMetaSearch.zip
then unzip it into the FARR plugins directory (usually C:\Program Files\FindAndRunRobot\Plugins) and restart FARR.

Usage

Type

    wms [search] [search term]
to use a pre-configured search.
